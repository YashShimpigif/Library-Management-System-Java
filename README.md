# Library Management System

Java-based Library Management System using Swing, JDBC, and SQLite.

## Features
- Book Management
- Issue & Return Books
- Late Fee Calculation
- SQLite Database

## Technologies
- Java
- Swing
- JDBC
- SQLite

## How to Run
1. Add sqlite-jdbc.jar to classpath
2. Compile using: javac *.java
3. Run using: java Main
